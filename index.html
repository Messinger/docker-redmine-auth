Docker-redmine-auth

Authentication server for Docker Registry 2 using redmine and/or gitlab service as backend

View on GitHub Download .zip Download .tar.gz

Docker registry Authentication Service against Redmine and Gitlab

Working:

  • docker-registry authentication against a redmine or gitlab installation
  • check access via repository pathes (push/pull allowed or not) or just login
  • a simple registry browser

Details about configuration etc see in wiki https://github.com/Messinger/docker-redmine-auth/wiki
